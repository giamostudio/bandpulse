What you get
Everything a DJ needs. Nothing else.
🥁
BPM Detection
Accurate tempo analysis using Essentia's multifeature algorithm. Same engine used by professional music analysis tools.
🎵
Key + Camelot
Musical key detection with Camelot Wheel notation. Know instantly which tracks mix harmonically.
⚡
Energy
Measure the intensity and density of a track. Build your set arc from intro to peak to closing.
💃
Danceability
ML-powered score that predicts how much a track will move a crowd. Powered by MSD danceability model.
🎛
Genre (Discogs)
Genre classification using 400 Discogs categories — from Deep House to Dub Techno to Funk to Ambient.
🔒
100% Offline
Everything runs on your Mac. Audio is never uploaded. No API keys, no accounts, no tracking.
Setup
Ready in 3 steps.
Start the server
Run once in Terminal and leave it open.

~/lupin_env/bin/python ~/Desktop/BANDPULSE/server/bandpulse_server.py

The server runs locally at localhost:5555 — nothing is exposed online.
Add the bookmarklet
Show your bookmarks bar with ⌘ Shift B, then drag the BANDPULSE button from this page into it. Takes 10 seconds.
Analyze on Bandcamp
Open any Bandcamp album or track page. Click the BANDPULSE bookmarklet. A panel opens showing every track with BPM, key, energy and genre. Analyze one or all.
🛡
Your music stays on your Mac.
Bandpulse downloads a short audio preview directly from Bandcamp to your machine, analyzes it with local ML models, and deletes the file immediately after. No data is ever sent to external servers. No analytics, no telemetry. The source code is open — verify it yourself.
Install
Free. No account needed.
See step-by-step slides →
1
Open Terminal on your Mac
Press ⌘ Space, type Terminal, press Enter.
Then paste this command and press Enter:
curl -fsSL https://raw.githubusercontent.com/giamostudio/bandpulse/main/install.sh | bash

First time only — takes ~2 minutes. After that, the server starts automatically at every login. No Terminal needed.

Show your bookmarks bar in Safari
Press ⌘ Shift B to show it if it's hidden.
